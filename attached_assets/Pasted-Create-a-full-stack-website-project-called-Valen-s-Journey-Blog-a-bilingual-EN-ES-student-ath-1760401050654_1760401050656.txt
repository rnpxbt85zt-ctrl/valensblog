### 📘 NOTION CMS SETUP
1. Create a Notion database with columns:
- **Title** (title)
- **Slug** (text)
- **Published** (checkbox)
- **PublishedAt** (date)
- **Summary** (text)
- **Cover** (files & media)
- **Content** (rich text)
2. Create a Notion integration at: https://www.notion.so/my-integrations
3. Copy the **Internal Integration Token** → Replit Secrets as `NOTION_TOKEN`
4. Copy your database ID → Replit Secrets as `DATABASE_ID`
5. Share the database with the integration (full access)
6. Server fetches only `Published = true` entries.
